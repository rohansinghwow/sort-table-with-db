# How to run this project

1. install all the packages

```
npm i
```

2. run this project

```
npm start
```

# Important Files

### db.js

- Used to provide creds and connect to DB

### server.js

- Express JS with all the routes and Postgres query

### DATA.json

- Contains mock data for offline testing
